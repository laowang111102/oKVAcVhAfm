# 购物管理系统简介 python436

## 项目概述


## 技术栈

- **后端语言**: Python
- **数据库**: MySQL

## 功能模块

- 用户管理：注册、登录等
- 商品信息管理：查看商品详情
- 举报记录管理：记录用户的举报信息
- 充值记录管理：管理用户充值行为
- 交流论坛：用户交流互动区域
- 订单管理：处理订单信息

## 系统角色

- 普通用户：注册、登录、查看商品、查看公告、在线留言
- 管理员：管理用户、商品、举报记录、充值记录、公告、订单等

## 运行环境

- Python环境
- MySQL数据库

## 部署步骤

1. 确保Python环境与MySQL数据库已正确安装
2. 配置数据库连接信息
3. 运行系统安装脚本，初始化数据库
4. 启动后端服务
